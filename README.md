## machine_learning_diary

**machine_learning_diary** 是 **人工智障工程师** 养成日记系列，代号简称**ML-Diary**。每次的课程以日记的形式发布，但不是日更的节奏。希望大家通过每天进步一点点，成为一个AI killer😝

每一个小日记，learn by doing，用来记录你在机器学习的点滴痕迹。

- 环境：Windows / MacOS
- 工具：Anaconda - Jupyter Notebook
- 版本：Python 3

## Why Girls-In-AI ？

GIA的理念就是，敲代码没有性别差异。只要你认为女生和男生一样能敲好代码就ok。

GIA是我写给我的朋友，和一些可爱的followers。我身边有很多朋友，未来是要学编程的，数据分析也好，技能存储也好，都关注了我，也给了我莫大的支持。但是她们现在考证的考证，考试的考试，都在自己的世界里拼搏着。我就在这里默默的给她们写教程。等她们准备好的那一天，等她们想学的那一天，不用思考是自学还是上培训班，不用纠结怎么辨别网上铺天盖地的课程质量，不用慌慌忙忙再去找。我想让她们知道，我永远在背后支持她们，以我自己的方式。如果你愿意，我就是你们那个敲代码的朋友。以我小小的力量，支持你们。



GIA不是教科书，整体的风格偏白话。我想让编程这件事简单一些，重在培养一些兴趣。已经有很多朋友反馈因为GIA敲下人生第一行代码，也因此对编程越来越感兴趣。如果我的坚持能带给你力量，那么我告诉你，我会一直坚持下去！

- [x] 编程零基础小白，这里有完整的学习路径，指导你从头开始掌握新技能
